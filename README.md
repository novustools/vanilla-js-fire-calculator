# Vanilla JS FIRE Calculator Logic

This repository contains the core, dependency-free JavaScript math logic for calculating Financial Independence, Retire Early (FIRE) metrics.

Built for developers who need lightweight, framework-free solutions. 

For the full, interactive enterprise B2B tool with visual charts, advanced reporting, and custom B2B print layouts, visit the [NovusTools FIRE Age Calculator](https://novustools.com/fire-age-calculator/).

## Usage
Simply integrate the logic and pass your financial variables (current age, target retirement age, savings rate, etc.) to get an instant calculation.
